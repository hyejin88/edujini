## 2. v2 비즈니스 모델 — Claude + Gemini 풀스택 자동화

### 2.1 콘텐츠 생산 — Gemini API 자동화

| 단계 | 모델 | 자동화 |
|---|---|---|
| 1차 생성 (단원당 100문항) | Gemini 2.5 Flash | 출판사 스타일 시스템 프롬프트 (천재·비상·미래엔 3종) |
| 검증 루프 (정답·KaTeX·라이선스) | Gemini 2.5 Pro | score ≥ 7만 통과 (~70%) |
| 변형 생성 (1 → 4~10) | Gemini Flash | 숫자·컨텍스트 변경 |
| 4축 오답 라벨링 (학생 풀이) | Gemini Flash | 호출당 ₩2 |
| 학부모 자연어 리포트 | Claude Sonnet 4.6 | 한국어 미세 톤 |
| 정혜진 샘플링 검수 | 본인 직접 | 5% (월 30분) |

→ **외주 검수 ₩9M 통째 제거**. 6주 안에 3,000문항 + 검증 + 라벨링.

### 2.2 개발 — Claude Code로 셀프

| 영역 | 도구 | 주 시간 |
|---|---|---|
| 백엔드 (FastAPI) | Claude Code | 10h |
| 프론트엔드 (Next.js + Tailwind + shadcn/ui) | Claude Code | 12h |
| DB·인프라 (Supabase) | Claude Code | 3h |
| 디자인 (shadcn/ui 베이스) | Claude Code | 0h (UI 라이브러리 활용) |
| QA·평가 | Claude (현재처럼 평가 스크립트) | 5h |

→ **풀스택 개발자 ₩7M/월 채용 비용 = 0**. 정혜진 주 30시간 투입.

### 2.3 라이선스 안전성 ↑

- Gemini로 자체 생성한 콘텐츠 = **자체 저작권 100% 귀속** (Google Gemini Terms)
- 시스템 프롬프트에 "DO NOT copy/paraphrase actual textbook" 명시
- 유사도 검사 (cosine similarity > 0.85 폐기) + 정혜진 샘플 검수
- 출판사 Rev-share는 **후순위로 미룰 수 있음** (Phase 2 이후 옵션)

## 10. 결론

**Claude Code + Gemini API 조합이 EduTech 1인 창업의 게임 체인저**입니다.

- **₩4,400만 → ₩440만**: 비용 1/9
- **9개월 BEP → 3~4개월 BEP**: 5배 빠른 손익분기
- **외주 검수·풀스택 채용 0**: 자기자본 ₩500만으로 시작 가능
- **콘텐츠 라이선스 리스크 완화**: Gemini 자체 생성 = 자체 저작권
- **14주 안에 베타 출시**: 빠른 PMF 검증

**바로 Claude Code 세션에서 백엔드 셀프 개발 시작 가능**. `backend/migrations/001_init.sql` Supabase 적용 → FastAPI 라우터 → Next.js UI → Gemini 통합 순서로 정혜진 + Claude 협업으로 14주 출시 가능.
