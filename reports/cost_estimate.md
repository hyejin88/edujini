## 5. Break-even 분석

### 가정 (Phase 2 — 6개월 차)
- 가입 유저: 10,000
- MAU: 5,000
- 유료 전환율: 5% → **유료 250명**
- 월 구독료: ₩9,900 (VAT 별도)
- 토스 수수료: 3%

### 월 매출 추정

| 시나리오 | 유료 유저 | MRR (KRW) | 연 환산 ARR |
|---|---|---|---|
| 비관 | 100 | 990,000 | 11.9M |
| 기본 | 500 | 4,950,000 | 59.4M |
| 낙관 | 1,000 | 9,900,000 | 118.8M |

### 월 운영비 (Phase 2 — 유료 500명 시)

| 항목 | 월 비용 |
|---|---|
| 인프라 (Vercel·Supabase·CDN) | 100,000 (트래픽 ↑) |
| LLM API (룰+캐시 유지 기준) | 500,000 |
| MathPix·기타 | 100,000 |
| 카카오 알림톡 (5,000건) | 50,000 |
| 토스 수수료 (매출의 3%) | 150,000 |
| 인건비 (개발 1 + 콘텐츠 0.5) | 9,000,000 |
| **합계** | **약 9,900,000** |

### Break-even Point

```
월 매출 ≥ 월 운영비
9,900,000 = 9,900,000
→ 유료 1,000명 시 BEP 도달
```

**현실적 도달 시점**: Phase 3 (9~12개월 차) — 가입 5만 / MAU 5천 / 유료 1,000 가정.

### 추가 매출 채널 (Break-even 가속)

| 채널 | 시점 | 예상 ARR |
|---|---|---|
| 진단 리포트 1회 결제 (₩9,900~29,000) | MVP+1mo | +30M~60M |
| 학원 B2B SaaS (좌석당 월 ₩1만 × 500좌석) | 6개월 차 | +60M |
| 출판사 Rev-share (간접 — 콘텐츠 비용 절감) | 9개월 차 | 비용 절감 |

→ B2B + 진단 리포트 결합 시 Phase 2 (6개월) 내 BEP 가시권.

## 7. 결론·권장

- **MVP 손익**: 무료 베타 동안 약 4,000만 원 비용 발생 (인건비 제외 1,600만 원)
- **BEP**: 유료 1,000명 시점 — 단순 B2C로는 9~12개월 소요
- **B2B 학원 SaaS 6개월 조기 도입**으로 BEP 시점 6개월 단축 가능
- **출판사 라이선스**가 가장 큰 외부 비용 변수 — 매출의 5~15% 양보 vs 콘텐츠 자체 제작 인건비 비교 → MVP는 **자체 제작 우선**, Phase 2부터 라이선스 검토

**권장 자금 조달**: 자기자본 5,000만 + 에인절 5,000만 = 1억 원으로 6개월 검증 + 진단 리포트 단일 결제 PMF 확인 → 시리즈 시드 본 라운드 진행.
