## 5. BEP 재계산 (v2)

### 가정
- 가입 유저: 1,000 (3개월 베타) → 10,000 (6개월) → 50,000 (12개월)
- 유료 전환율: 5%
- 월 구독료: ₩9,900

### Phase 2 (6개월 차) 월 운영비

| 항목 | 월 (KRW) | 비고 |
|---|---|---|
| Vercel Pro | 27,000 | 트래픽 증가 |
| Supabase Pro | 33,000 | 유저 증가 |
| Cloudflare R2 | 30,000 | 5만 가입 |
| Gemini API | 1,500,000 | 학생 5,000명 풀이 라벨링·생성 |
| Claude API | 200,000 | 학부모 리포트 5,000명 |
| 카카오 알림톡 | 50,000 | 5,000명 |
| 토스 수수료 | 75,000 | 매출의 3% |
| **합계** | **약 ₩1,915,000** | 인건비 0 |

### Break-even Point

```
월 매출 ≥ 월 운영비
0 인건비 시: 1,915,000 / 9,900 ≈ 194명
```

**유료 200명 = BEP**. 가입 4,000 / 유료 전환율 5% 시 도달 가능 → **3~4개월 차 BEP 가시권**.

→ v1 BEP 1,000명 (9~12개월) → **v2 BEP 200명 (3~4개월)** = **6~8개월 단축**.
