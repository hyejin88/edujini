안녕하세요, 천재교육 디지털사업본부 사업개발팀 담당자님.

[회사명]에서 신규 EduTech 웹앱 **"Edujini"**(가제)를 준비 중인 정혜진입니다. 천재교육의 교과서·디지털 학습 자산이 한국 K-12 시장에서 보유하신 커버리지를 깊이 신뢰하고 있어, 협업 제안을 드리고자 메일드립니다.

## 1. Edujini 서비스 개요

- **컨셉**: 초1~고3 모든 교과서 단원별 기출·연습문제 + AI 오답 4축 진단(개념/계산/해석/함정) + 약점 추천 학습
- **차별점**: 학년·교과 가로지르는 단원 그래프 (분수 약점이 어느 학년·단원에서 시작됐는지 역추적)
- **타깃**: B2C 학생·학부모 (1차) → 학원 SaaS (Phase 2)
- **MVP 출시 예정**: 2026년 7월 (중1 수학 단일 과목으로 시작)

## 2. 천재교육 협업 제안 — 매출 분배 라이선스

천재교육 교과서 콘텐츠(평가문제·해설·단원 가이드)를 본 서비스 내에 합법적으로 활용하기 위해, 다음 형태의 라이선스 협의를 제안드립니다.

| 항목 | 제안 |
|---|---|
| **라이선스 범위** | 천재교육 검정 교과서(2022 개정) 기반 평가문제 + 단원 가이드 — 디지털 사용권 (재가공·재배포 X) |
| **수익 분배** | 라이선스된 콘텐츠가 노출된 풀이 세션 매출의 **5~15%** Rev-share (협의) |
| **데이터 보고** | 월 단위 단원별 노출·풀이·정답률 리포트 자동 제공 → 천재교육 콘텐츠 개선에 역활용 가능 |
| **브랜드 노출** | 풀이 화면 하단에 "천재교육 [교과서명] 출처" 배지 표시 |
| **공동 마케팅** | T셀파·밀크T 사용자 대상 Edujini 진단 무료 체험 링크 제공 가능 |

## 3. 천재교육 측 기대 효과

- **신규 매출 채널**: 디지털 학습 시장에서 자체 플랫폼(밀크T) 외 추가 수익 확보
- **콘텐츠 활용도 ↑**: 정형화된 풀이 데이터로 콘텐츠 품질·난이도 개선 인사이트
- **AI 디지털교과서(AIDT) 활용률 8.1% 좌초** 이후 디지털 학습 시장 재편 시점 — 작고 빠른 파트너십으로 위험 분산
- **B2C 채널 확대**: 학원·B2B 영업 비용 없이 학부모·학생에게 직접 도달

## 4. 다음 스텝 제안

1. **킥오프 미팅** (30분) — 양사 비전·범위 정렬 (5월 둘째 주 희망)
2. **NDA 체결 후 시범 라이선스** — 중1 수학 1종 (단원 가이드 + 평가문제 1,000건)으로 3개월 테스트
3. **데이터 검토 후 확대** — 효과 확인 시 중2·중3 + 국·영 + 고등 단계적 확장

기존 천재교육의 자체 플랫폼(밀크T·T셀파)과 직접 경쟁하지 않는, **외부 진단·약점 분석 레이어**로 포지셔닝하여 시너지 구조를 만들고자 합니다.

답변 주시면 자료 첨부와 미팅 일정을 제안드리겠습니다.
