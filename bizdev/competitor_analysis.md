# EduQA 경쟁사 심층 분석 보고서

> **작성일**: 2026-05-01  
> **대상 서비스**: EduQA — 초1~고3 NCIC 성취기준 기반 모바일 문제집 + AI 4축 오답 진단  
> **목적**: 1차 조사된 3개사(일일수학·매쓰플랫·족보닷컴) 외 10개 경쟁사 심층 분석 후 EduQA 포지셔닝·차별 메시지·가격 검증·위협 분석을 산출  
> **시장 규모 참고**:
> - 2024년 한국 초중고 사교육비 총액 **29조 2,000억원** (YoY +7.7%)
> - 학생 1인당 월평균 사교육비 **47.4만원** (초 44.2만원 / 중 49만원 / 고 52만원)
> - 2026년 한국 에듀테크 시장 약 **11조원** 전망
> - K-12 사교육 참여율 **80.0%** (초 87.7% / 중 78.0% / 고 67.3%)

## 4. EduQA 포지셔닝

### 4-1. 시장에서 EduQA가 끼는 자리

```
                  [고가 종합학습 (홈런·밀크T·와이즈캠프 — 월 7만~14만)]
                                  │
                                  │
[B2B 학원 (매쓰플랫·매쓰홀릭)]    │   [출판사 직영 (천재·비상·미래엔)]
                                  │
                  ─────[ EduQA ]─────
                       (₩990~9,900)
                       모바일 + AI + 전학년
                                  │
[검색 도구 (콴다)]                │   [공교육 무료 (똑똑수학·EBS)]
                                  │
                                  │
                     [무료 PDF (일일수학·기출)]
```

EduQA는 **"고가 종합학습기와 무료 PDF 사이의 빈 공간"** + **"검색도구(콴다)와 문제집 사이의 빈 공간"** 두 개를 동시에 차지하는 **하이브리드 포지션**.

### 4-2. 직접 경쟁자 vs 보완재

| 분류 | 서비스 | 이유 |
|---|---|---|
| **직접 경쟁** | 콴다 프리미엄 (월 18.5k) | 같은 B2C × 모바일 × AI × 학생 직접결제 슬롯. 단, 콴다는 "검색", EduQA는 "문제집·진단" → **타겟 use case가 다름** (콴다=숙제할 때, EduQA=시험 대비할 때) |
| **직접 경쟁** | 족보닷컴 (중·고) | 중1~고3 시험 대비 정면충돌. EduQA가 NCIC 성취기준 + AI 변형 + 4축 진단으로 차별화 필요 |
| **간접 경쟁** | 일일수학 | 초등 연산만 — EduQA가 전과목·전학년이라 보완재성 강함. 단, **"무료" 기대값을 학부모에게 학습시킨 적** |
| **간접 경쟁** | 똑똑수학탐험대 | 초등 수학 무료. EduQA가 중·고 + 4축 진단 + 학부모 리포트로 우위 |
| **상위 시장** | 홈런·밀크T·와이즈캠프·엘리하이 | 월 7만~14만 종합학습기. EduQA는 **"종합학습기 검토 전 ₩990 워밍업"** 포지션 |
| **B2B 시장** | 매쓰플랫·매쓰홀릭·클래스팅 | EduQA의 B2C 직판 + 학부모 카톡 리포트와 거의 충돌 안 함 |
| **보완재 (얼라이언스 가능)** | EBSMath, 똑똑수학탐험대 | 무료 학습 후 "더 풀고 싶은 학부모"를 EduQA로 전환 가능 |

### 4-3. EduQA만 가진 6가지 조합 (시장 유일)

1. **초1~고3 전학년 단일 앱** — 형제자매 한 앱으로 커버 (홈런·밀크T·와이즈캠프는 학년급별 분리 구매)
2. **AI 동적 생성 + 변형 무제한** — 매쓰플랫은 B2B, 콴다는 검색 → EduQA가 B2C × 변형 생성 유일
3. **출판사 스타일 모방** (천재·비상·미래엔) — 학부모가 학교 교과서랑 똑같다고 느낌
4. **4축 오답 라벨링** (개념미숙 / 계산실수 / 문제해석 / 함정미인지) — 어느 경쟁사도 명시 라벨 없음
5. **학부모 카톡 리포트 (Anthropic/Gemini 자연어)** — 매쓰플랫 SMS, 콴다 없음, 종합학습기 코칭매니저 콜
6. **₩990 단발 + ₩9,900 월구독** — 시장 최저가 진입로. 충동구매 가능 가격대

### 7-2. 시나리오별 위협

**시나리오 A — 콴다가 "AI 문제집 모드" 추가** (가장 큰 위협)
- 매스프레소가 콴다 프리미엄에 "내 단원 풀이세트" 기능 추가하면 EduQA 정면 잠식
- 9,700만 글로벌 가입자, 한국 중·고생 50% 점유 → 유통력 압도
- **방어책**: ① **NCIC 성취기준 매핑 + 출판사 스타일 모방** 깊이 — 콴다는 글로벌 우선이라 한국 교과 디테일에 약함 ② **학부모 카톡 리포트** — 콴다는 학생 단독 도구 ③ **₩990 단발** — 콴다는 월 구독만

**시나리오 B — 족보닷컴이 AI 변형문제 도입** (2순위 위협)
- 4,800개 학교 내신 기출 + AI 변형 = "학교 시험에 그대로 나올 문제 무한 생성" 가능
- **방어책**: ① **초등까지 커버** (족보닷컴은 중·고만) ② **4축 오답 라벨링** — 족보닷컴은 정답·해설만 ③ **모바일 즉시 풀이 UX**

**시나리오 C — 클래스팅 AI가 학교 단체 보급 확장** (3순위 위협)
- 4,600개 학교 → 8,000개로 늘면 학생 다수가 "학교에서 깐 클래스팅"으로 EduQA 필요성 감소
- **방어책**: ① **학교 외 시간(시험 직전)** 사용 시나리오 강화 ② 학부모 직접결제로 클래스팅과 채널 분리 ③ **시험 D-7 단원 정복** 같은 단발 상품 강화

**시나리오 D — 종합학습기 (홈런·밀크T) 가격 인하** (낮은 위협)
- 월 7~14만 → 월 3~5만으로 내려와도 여전히 EduQA의 3~5배 → **즉각적 위협 아님**

### 7-3. 1순위 위협 콴다에 대한 EduQA 방어 4단계

1. **속도** — 콴다가 "AI 문제집 모드"를 만들기 전에 학부모 카톡 리포트와 ₩990 단발로 먼저 침투
2. **깊이** — NCIC 성취기준 + 출판사 라이선스(천재·비상·미래엔 협의) → 콴다 글로벌 코드베이스로 못 따라옴
3. **틈새** — 콴다가 약한 **초1~6 + 학부모 리포트** 영역에 자원 집중
4. **얼라이언스** — EBSMath·똑똑수학탐험대 무료 사용자에게 "더 풀고 싶으면 EduQA ₩990" 깔때기 만들기

## 8. 액션 권고 — EduQA가 살아남기 위해 무엇을 해야 하는가

### 8-1. 즉시 (0~1개월)

1. **출판사 라이선스 협의 시작** — 천재·비상·미래엔 — "스타일 모방"이 카피 vs 라이선스 회색지대이므로 법적 리스크 우선 정리 (이미 작성된 `bizdev/license_email_*.md` 적극 활용)
2. **₩990 단발 결제 UX 완성** — saju-kid.com 벤치마크. 회원가입 없이 카카오페이 원클릭 → 단원 1개 락 해제
3. **랜딩 페이지 4축 진단 데모** — "내 아이가 왜 틀렸는지 30초 만에 보여드립니다" 인터랙티브 데모 → 60문항 무료 가입 전환
4. **콴다 비교 카피 명확화** — "콴다는 모를 때 찍는 도구, EduQA는 시험 대비 문제집"

### 8-2. 단기 (1~3개월)

5. **학부모 카톡 리포트 베타** — 주 1회 자동 발송 + 자연어로 "이번 주 ○○이는 분수 단원에서 함정미인지 3번"
6. **시험 D-7 캠페인 패키지** — 초·중·고 시험기간(중간/기말)에 ₩990 단발을 캠페인성 푸시
7. **NCIC 성취기준 매핑 공개** — 단원별 성취기준 코드(예: 4수01-05)를 노출 → 학부모 신뢰
8. **무료 60문항 → ₩990 전환율 측정** (북극성 지표). 목표 8% 이상

### 8-3. 중기 (3~6개월)

9. **연간 결제 ₩89,000 출시** — LTV 확보, OTT 수준 가격대 강조
10. **얼라이언스** — 똑똑수학탐험대·EBSMath 사용자 대상 제휴 마케팅 (무료 → 유료 깔때기)
11. **학교 단체 OEM 모드** — 클래스팅 위협 방어용 — 학교가 EduQA를 단체 도입할 수 있는 라이선스 트랙 (B2G 백업 채널)
12. **AI 4축 진단 정확도 외부 검증** — 1,000명 베타 데이터로 논문/보도자료 — "함정미인지 진단 정확도 92%"

### 8-4. 장기 (6~12개월)

13. **고등 모의고사 변형 시즌제** — 수능 D-100 캠페인 → ₩9,900 월구독 시즌 가입자 폭증 유도
14. **학부모 커뮤니티** — 카톡 리포트 → 학부모 후기·재구매 → NPS 60 이상 확보 → 콴다·족보닷컴이 못 따라오는 락인
15. **간이과세 → 일반사업자 전환 시점 결정** — 매출 4,800만/년 도달 시 부가세 환원 + 법인 전환 검토
16. **출판사 정식 파트너십** — 라이선스에서 공동 마케팅으로 — 천재교과서 밀크T가 약한 "변형 무제한" 영역을 EduQA가 커버하는 그림

## 9. 결론 — 한 줄 요약

> **"₩9,900 + AI 변형 + 4축 진단 + 카톡 리포트"는 시장 진공지대다. 단, 콴다가 같은 모델로 진입하기 전에 NCIC 깊이·출판사 라이선스·학부모 카톡 락인으로 12개월 안에 해자를 만들지 못하면 자본력에 밀린다. ₩990 단발은 충동구매 임계 아래라 그대로 유지하고, 무료 60문항 → ₩990 전환율을 북극성 지표로 잡아라.**
