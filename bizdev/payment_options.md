## 2. 추천 시나리오

### 시나리오 A. **Lemon Squeezy 단독** (1순위)
- ✅ MoR로 사업자 불필요
- ✅ 5% 수수료 (Gumroad 10%보다 저렴)
- ✅ 정기결제 / 트라이얼 / 환불 / 인보이스 자동화
- ⚠️ 카카오페이·네이버페이 미지원 → 한국 학부모 카드 결제만 가능
- ⚠️ 정산: USD → Wise → 한국 통장 (환율·송금 수수료)

**적용 가격(임시) 산식 (₩4,900/mo 가정, USD 환산 $3.50):**
- 매출 $3.50 → LS 수수료 5% + $0.50 = $0.68 → 정산 $2.82 (₩3,950)
- 실효 수령률 ≈ **80.6%**

### 시나리오 B. **Lemon Squeezy + 페이히어 병행** (2순위)
- 해외 사용자 + 카드 결제 → Lemon Squeezy
- 한국 학부모 + 카카오페이 → 페이히어 (개인사업자 등록 후)
- 단점: 정혜진이 **개인사업자 등록 필요** → 사업자 등록 회피 원칙과 충돌

### 시나리오 C. **Gumroad 단독** (3순위, 빠른 실험용)
- ✅ 가장 빠른 셋업 (5분, SKU만 만들면 끝)
- ❌ 수수료 10% → 매출 ₩4,900 → 정산 ₩4,109 (수령률 83.8%) ※ LS 보다 약간 낮음
- ❌ 정기결제 UX 가 LS 대비 떨어짐
- 장점: PoC 단계 빠른 검증에 유리

### 시나리오 D. **Buy Me a Coffee 후원 모델** (보조)
- "정기 후원 멤버십 ₩3,000/mo" 형태
- 수익화라기보단 응원 수단
- LS와 **병행**하여 가벼운 진입을 원하는 사용자에게 옵션 제공

## 3. Edujini 채택 결정

### 🥇 1순위: **Lemon Squeezy 단독**
**이유**:
1. 사업자 미등록 원칙 유지
2. 수수료 최저 (5%)
3. 정기결제 + 환불 + 인보이스 자동화 → 1인 운영 부담 0
4. MoR가 부가세·VAT 처리 → 세무 리스크 0
5. Gumroad보다 결제 페이지 UX 우수

### 🥈 2순위 (병행 검토): **Buy Me a Coffee**
- 후원형 결제 채널 추가 → 정기결제 거부감 있는 사용자 흡수
- 별도 SKU 발행 부담 없음

### ⏳ 미래 옵션: **페이히어 (개인사업자 등록 후)**
- 매출 750만/년 초과 시 개인사업자 등록 자동 트리거
- 카카오페이·네이버페이 도입으로 한국 학부모 UX 대폭 개선
- 시점: Stage 4 이후

## 4. 운영상 주의사항

### 정산 흐름 (Lemon Squeezy 채택 시)
1. 사용자 결제 → LS Stripe 처리
2. LS가 수수료·VAT 차감 후 USD 잔액 누적
3. 매주 또는 월 1회 → PayPal 또는 Wise 계정 송금
4. Wise → 한국 농협/카카오뱅크 계좌
5. 환율·송금 수수료 약 1~2% 추가 손실

### 세무 처리 (개인 기준)
- LS는 MoR이므로 정혜진은 **콘텐츠 공급 수익**으로 간주
- 한국 소득세법상 **기타소득**(지속·반복 X) 또는 **사업소득**(지속·반복 O) 분류
- Edujini는 지속·반복 → **사업소득** → 종합소득세 신고 의무 (사업자 등록과 별개)
- 연 750만원 이하 → 분리과세 옵션 가능
- **결론**: 사업자 등록은 안 해도 되나 종합소득세 신고는 매년 5월 필수

### 환불·분쟁 처리
- LS 환불 정책 따름 (기본 7일)
- 분쟁 발생 시 LS가 1차 응대, 정혜진 개입 최소화
- DMCA 또는 저작권 분쟁 발생 시 LS는 콘텐츠 호스팅 X → 정혜진이 직접 대응
