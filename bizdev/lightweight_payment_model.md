1. **싸다** — ₩990 = 카페 음료 1잔보다 저렴, 충동구매 가능
2. **빠르다** — 카카오페이 1탭 결제, 회원가입 X
3. **즉시 가치** — 결제 후 1분 안에 결과 (PDF·웹 페이지)
4. **단발성** — 정기결제 부담 X, 매번 새 상품을 내가 선택

→ 정혜진이 표현한 **"쉽게 접근하고 쉽게 결제"** 의 정확한 모델.

## 4. 사업자 등록 진짜 비용 재계산

정혜진이 "사업자 등록 안 하고 싶다" 의지를 표현했지만, 본질은 **"행정·세무 부담 회피"**일 가능성이 높음. 실제 등록 부담을 정확히 측정해보자:

### 사업자 등록 (간이과세자) 실제 부담

| 항목 | 실제 |
|---|---|
| **등록 절차** | 세무서 방문 또는 홈택스 → **15분, 무료** |
| **기간** | 1~3일 발급 |
| **간이과세자 자격** | 연 매출 1억 미만 + 일부 업종 (정보통신·교육 OK) |
| **부가세** | 연 매출 4,800만 미만 → **부가세 면제** |
| **종합소득세** | 5월 신고 (사업자 등록 여부와 무관, 어차피 신고해야 함) |
| **유지비** | 0원 |
| **폐업** | 홈택스에서 5분, 무료 |

### 결론
- 사업자 등록 자체는 **15분 + 무료**
- 부가세 부담은 매출 4,800만 미만이면 **0**
- 종합소득세는 **사업자 등록과 무관** (사업소득은 어차피 신고)
- 통신판매업 신고는 **추가** (별도, 매출 발생 후 30일 이내)

→ **"사업자 등록"의 실질 부담은 거의 0**. 정혜진이 표현한 "사업자 안 함"의 본질은 **"법인 세팅·세무대리인·복잡한 회계"**일 가능성이 높음. 간이과세 개인사업자는 그 부담이 거의 없음.

## 5. 의사결정 옵션 (정혜진 선택 필요)

### 옵션 A. 사업자등록 X 유지 (현재 의지 유지)
- Gumroad 또는 Lemon Squeezy + Buy Me a Coffee
- **카카오페이 X** → 한국 학부모 진입 어려움
- ₩990 단발 모델은 **사실상 운영 불가**
- 대신 ₩4,900~9,900 카드 단발/구독만 가능
- 매출 천장: 한국 시장의 30% (해외·재외동포 + 카드 익숙 학부모)

### 옵션 B. 간이과세 개인사업자 등록 (15분 투자)
- 부트페이 또는 페이히어 → **카카오페이 1탭**
- ₩990 단발 결제 가능 → **saju-kid 스타일 그대로 구현**
- 매출 4,800만/년까지 부가세 면제
- 종합소득세는 어차피 신고 의무
- 통신판매업 신고만 추가 (5분)
- 매출 천장: **한국 시장 100%**

### 옵션 C. 하이브리드
- 1차: 사업자 X + Gumroad ₩4,900 단발 (기능 검증)
- 2차: 매출 가시화 + 사용자 요청 → 사업자 등록 + 카카오페이
- 단점: 1차에서 진입장벽으로 KPI 왜곡
