[회사명]에서 신규 EduTech 웹앱 **"Edujini"**를 준비 중인 정혜진입니다. 비상교육이 보유한 초·중·고 수학 교과서 시장 점유율과 OnlyOne·비바샘 디지털 자산을 깊이 신뢰하고 있어, 협업 제안 메일드립니다.

## 1. Edujini 핵심 개요

- **컨셉**: 초1~고3 모든 교과서 단원별 기출·연습문제 + AI 오답 4축 진단(개념/계산/해석/함정) + 약점 추천 학습
- **차별점**: 학년·교과 가로지르는 단원 그래프 + 선수학습 의존성 역추적 (콴다=수학·홈런=초등 등 경쟁사 모두 편중 — 시장 빈틈)
- **타깃**: B2C 학생·학부모 우선
- **MVP 출시 예정**: 2026년 7월 — 중1 수학 단일 과목 시작

## 2. 비상교육 협업 제안

비상의 강세 영역인 **수학·영어·과학** 교과서 콘텐츠를 Edujini 내 합법적으로 활용하기 위해 다음 형태의 라이선스 협의를 제안드립니다.

| 항목 | 제안 |
|---|---|
| **1차 라이선스 범위** | 비상교육 중1·중2·중3 수학 교과서(2022 개정) 평가문제 + 단원 가이드 |
| **수익 분배** | 라이선스된 콘텐츠 노출 세션 매출의 **5~15%** Rev-share |
| **데이터 보고** | 월 단위 단원별 풀이·정답률·약점 분포 리포트 → 비상 콘텐츠팀 개선 인사이트 |
| **브랜드 노출** | 풀이 화면 출처 배지 ("비상교육 [교과서명] 출처") |
| **OnlyOne 시너지** | OnlyOne 학습자에게 Edujini 진단 무료 체험 제공 — 비상 자체 채널 강화 |
| **B2B 확장 시** | 학원 SaaS 출시 시 비상 콘텐츠 우선 노출권 부여 |

## 3. 비상교육 측 기대 효과

- **수학 교과서 1위 자산의 디지털 채널 확장**: 자체 OnlyOne 외 추가 수익 채널
- **AIDT 좌초 이후 시장 재편**: 작은 EduTech 파트너십으로 외부 트렌드 흡수 — 위험 분산
- **데이터 인사이트**: 단원별 학생 약점 분포 데이터를 콘텐츠 개정·교사용 자료에 역활용
- **학원 영업 부담 X**: B2C 채널로 학생·학부모 직접 도달

## 4. 비상 vs 다른 출판사

[회사명]은 동시에 천재교육과도 라이선스 협의 진행 예정이지만, **수학 영역 1차 협업 파트너로 비상교육을 우선 검토**하고 있습니다. 사유:
- 비상의 수학 교과서 시장 점유율
- OnlyOne 자체 플랫폼이 학습지 패드 중심 → Edujini 모바일 웹과 채널 충돌 적음
- 출판사 데이터 인사이트 협력 모델이 양사 모두에 유리
