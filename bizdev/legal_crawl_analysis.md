## 1. 적용 법령 요약

### 저작권법
- **§4** 저작물 정의 — 인간의 사상·감정의 창작적 표현물
- **§24-2** 공공저작물 자유이용 — 국가·지자체 작성 저작물은 영리목적 포함 자유이용 가능
- **§28** 공표저작물 인용 — 보도·비평·교육·연구 목적 정당범위 내 인용 가능
- **§35-3** 공정이용 — 4요소 테스트 (① 이용목적·성격 ② 저작물 종류·용도 ③ 이용 부분·비중 ④ 시장 영향)
- **§101** 저작권 침해죄 — 5년 이하 징역 또는 5천만원 이하 벌금 (친고죄, 영리·상습은 비친고)

### 정보통신망법 §50 (크롤링 관련)
- robots.txt 위반은 직접적 형사처벌 대상은 아니나 영업방해죄·업무방해죄 적용 가능 (대법원 2017도10546)
- 과도한 요청으로 서버 부하 유발 시 형법 §314 업무방해죄

### 부정경쟁방지법 §2-1 카목
- 타인의 상당한 노력으로 만든 성과를 무단 이용하여 경제적 이익 침해 → 민사 손해배상

## 3. 안전 출처 상세 분석

### 3-1. 시·도교육청 학력평가
- **출처**: 서울시교육청, 경기도교육청, 부산시교육청 등 17개 시·도
- **저작권**: 공공저작물 (§24-2). 영리 이용 가능
- **수집 가능 항목**: 본문 / 정답 / 해설 / 출제 단원 / 출제 의도
- **수집 방법**: 각 교육청 공식 홈페이지 PDF 다운로드 → OCR → 구조화
- **연간 산출량**: 학년당 학력평가 약 2~4회 × 17개 시도 = 학년당 30~60문항 × 과목
- **EduQA 적용**: 초3~고3 전 학년 시드 데이터로 활용

### 3-2. 한국교육과정평가원 (KICE)
- **수능 본시험**: 매년 11월. 공공저작물.
- **6월·9월 모의평가**: 공공저작물.
- **수집 경로**: KICE 공식 홈페이지, 한국교육과정평가원 자료실
- **EduQA 적용**: 고3 대비 핵심 콘텐츠

### 3-3. NCIC (국가교육과정정보센터)
- **성취기준 코드**: `[4수01-01]` 형식 (예: 초4 수학 1단원 1번)
- **자유이용**: 공공저작물 + KOGL Type 1 (출처표시 조건만)
- **EduQA 적용**: 모든 자체 생성 문항의 태깅 기준

## 4. 자체 생성 문항의 저작권

### 4-1. Gemini가 생성한 문항의 저작권자
- **Google Gemini API Terms (2025 버전)**: 사용자(EduQA)가 입력한 프롬프트 결과물의 IP는 **사용자에게 귀속**.
- **단**: 다른 저작물을 무단 학습한 결과 생성된 문항이 원본과 실질적으로 유사할 경우 → 저작권 침해 위험 존재.

### 4-2. 침해 회피 가이드라인 (시스템 프롬프트 반영 완료)
```
DO NOT copy or paraphrase any actual textbook problem you may have seen.
DO NOT reference actual 천재교육 trademarked names.
```
- 출판사 **스타일**만 모방 (서술톤, 구성, 보기 패턴) → 표현 자체는 자체 생성
- 검증 단계(Pro)에서 "no copyright concerns" 항목 점검 (이미 `VERIFY_SYSTEM`에 포함)
- 사후 모니터링: 베타 사용자가 "이거 OO 출판사 문제 같은데" 신고 시 → 즉시 회수 + 재생성

## 5. 크롤링 기술 가이드라인

### 5-1. 안전 패턴
1. **robots.txt 준수** — 각 도메인 robots.txt 확인 후 disallow 경로 회피
2. **User-Agent 명시** — `EduQA-Bot/0.1 (contact: 정혜진 이메일)`
3. **Rate Limit** — 도메인당 1 req/sec 이하
4. **공식 API 우선** — KICE, 교육청 OpenAPI 제공 시 그쪽 사용
5. **PDF 직접 다운로드 후 로컬 OCR** — 서버 부하 최소화

### 5-2. 위험 패턴 (절대 금지)
- ❌ 출판사 사이트 회원가입 → 콘텐츠 수집
- ❌ 학습 플랫폼(콴다·아이스크림 등) 본문 스크래핑
- ❌ 카페·블로그에 업로드된 출판사 문제 수집 (출판사 저작권 그대로 유지)
- ❌ Cloudflare·접근 제한 우회

## 7. 결론

### ✅ 안전하게 활용 가능 (총 약 5,000~10,000문항/년)
- NCIC 성취기준 (메타)
- 시·도교육청 학력평가 (전 학년 약 1,000문항/년)
- 평가원 수능·모의평가 (고3 약 200문항/년)
- 자체 생성 문항 (출판사 스타일 모방, 본문 자체 작성, 무제한)

### ⚠️ 회색지대 (별도 가이드라인 필요)
- EBS 교재 — 메타데이터만 활용, 본문 X
- 사설 학원 무료 공개 자료 — 명시적 무료배포 라이선스 확인 필수

### ❌ 절대 회피
- 출판사 교과서·문제집 본문 복제
- 사설 모의고사 본문 복제
- 회원제 플랫폼 스크래핑

### 🎯 EduQA 콘텐츠 전략
**80%** 자체 생성 (Gemini, 출판사 스타일 모방) + **15%** 공공 기출 (교육청·평가원) + **5%** EBS 메타 큐레이션 (본문 외부 링크).
이 구조는 저작권 분쟁 발생 가능성이 매우 낮으며, 1인 운영 + 사업자 미등록 환경에서도 운영 리스크 통제 가능.
