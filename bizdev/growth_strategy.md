## 1. 타깃 페르소나 (1순위)

### P1 — 초3~6 자녀 학부모 (핵심)
- 30~45세 여성, 자녀 1~2명
- 사교육비 월 30~50만원 부담
- "학원 + 학습지 사이의 무료 보강" 검색
- 채널: 맘카페 (네이버), 인스타그램 릴스, 카카오스토리, 블로그
- 의사결정: 자녀 학습 효과 > 가격 > 디자인 > 브랜드

### P2 — 중·고등 학생 본인
- 13~18세, 자기 주도 학습
- "기출문제 무료", "수학 단원 정리" 검색
- 채널: 유튜브, 디시·에타, 콴다 대안 검색
- 의사결정: 콘텐츠 양 > 정답·해설 정확도 > 인쇄 편의

### P3 — 학습 코치·과외 선생님 (보조)
- 자료 제작 시간 단축 목적
- 학생용 학습지 인쇄 후 활용
- 채널: 네이버 카페 "과외선생님 모임", 페이스북 그룹

## 2. 채널 전략 (12주 로드맵)

### Week 1~2 — 셋업
- AdSense 신청 (도메인·privacy 페이지·콘텐츠 30+ 페이지 확보)
- Google Search Console 등록 + 사이트맵 제출
- Naver Search Advisor 등록
- 카카오톡 채널 개설 (1:1 문의)
- privacy / terms / 운영자 페이지 추가 (AdSense 정책 준수)

### Week 3~4 — 시드 100명
- **정혜진 SNS 직접 공유** (페북·인스타·카카오스토리)
- 지인 학부모 베타 초대 5~10명
- 첫 후기 캡처 → 트러스트 quote 갱신
- 유튜브 1~2분 사용법 영상 1개

### Week 5~8 — 확산 1,000명
- **맘카페 게시물** 5~10개
  - 네이버 "엄마들의 수다", "맘스홀릭", "베이비페어맘"
  - 광고성 X, 후기 형태로
  - 제목 예: "초3 수학 학습지 무료로 푸는 방법 찾았어요"
- **블로그 SEO 30개 포스트**
  - "초3 1학기 수학 단원평가 무료" (롱테일)
  - "초3 분수와 소수 학습지 PDF"
  - "AI 자동 채점 수학 학습지"
  - 각 단원별 1개씩 (수학 12단원 × 학년 6 = 72개 잠재)
- **인스타그램 릴스** 주 2개
  - 학습지 인쇄 → 풀이 → 자동 채점 화면 녹화
  - BGM 무료 라이브러리

### Week 9~12 — 1,000 → 5,000 명
- **유튜브 채널 시작** (정혜진 본인 또는 자녀 풀이 영상)
- **카카오톡 오픈채팅** "초등 수학 학습지 공유방"
- **Google Ads** 키워드 입찰 (필요 시 ₩100,000/월 테스트)
- **인플루언서 협업** — 마이크로 인플루언서 (팔로워 1만~3만 학부모) 1~2명 시제품 사용

## 7. 광고 정책 (자체 규율)

### 절대 광고 X 페이지
- 학습지 (`/library/[unitId]`) — 학습 신뢰감
- 진단 결과 (`/result`)
- 학부모 리포트 (`/parent-preview`)

### 광고 노출 제한
- 라이브러리 페이지 1개 (rectangle, max-w-sm, 푸터 직전)
- 메인 광고 X (첫인상 깔끔)

### AdSense 컴플라이언스
- "Made for Kids" 사이트 등록 → 비개인화 광고만
- 콘텐츠 중간 광고 X (학생 풀이 흐름 끊지 않음)
- 우발 클릭 유도 디자인 X (배너·박스 명확 분리)
