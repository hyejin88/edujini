## 1. 핵심 가치 제안 (UVP)

> "**초1부터 고3까지, 모든 교과서의 약점이 어디서 시작됐는지 역추적해 줍니다.**
>  AI 오답 진단 + 학년·단원 그래프 + 학부모 주간 리포트."

차별점:
- 단순 정답 채점이 아닌 **오답 유형 4축 라벨링** (개념/계산/해석/함정)
- 학년·교과 가로지르는 **단원 그래프 + 선수학습 의존성** (콴다=수학 / 홈런=초등 / 시매쓰=수학 등 모든 경쟁사 편중)
- **결제자(학부모) ≠ 사용자(학생)** 분리 UX

## 4. 핵심 사용자 플로우

### F1. 신규 가입 → 첫 진단
```
랜딩 → 회원가입(소셜)
→ 학년·교과서 선택
→ 진단고사 10문항 (난이도 자동 조절, 5분)
→ 약점 리포트 + 추천 5문제
→ 무료 사용 시작 (일 5문제 무료)
```

### F2. 일일 학습 루프
```
홈 → 추천 5문제
→ KaTeX 풀이 (객/단답)
→ 즉시 채점 + 해설
→ 오답 유형 라벨링 (자동) + 후속 추천
→ 스트릭 +1 / 진척도 갱신
```

### F3. 주간 학부모 리포트 (재구독 핵심 레버)
```
일요일 21시 자동 배치
→ "이번 주 OO이는 분수 약점이 다시 노출됐어요"
→ 카카오 알림톡 발송
→ 학부모 대시보드 링크 → 결제 갱신 게이트
```

## 6. AI 진단 엔진 — 3축 결합 추천

```
약점 점수    × 0.5  (mastery 하위 20% 개념 우선)
난이도 적합도 × 0.3  (현재 θ ± 0.3, 정답률 60~75% 타겟)
간격 반복   × 0.2  (SM-2 변형: 1일/3일/7일/21일)
```

상위 10개 후 **MMR 다양화 재정렬** → 5개 추천.

## 7. 콘텐츠 파이프라인 (라이선스 안전)

```
[Source]
├─ EBS 공공 자료 (무료 라이선스)
├─ 시도교육청 학력진단 (협의)
├─ 자체 제작 (성취기준 [4수01-01] 코드 기반)
└─ AI 변형 문항 (자체 제작 시드 → LLM 변형 → 검수자)

[Pipeline] (Airflow DAG)
OCR (MathPix) → 단원 매핑 (LLM) → 검수자 어드민 → DB 적재
                         ↓
                    개념 태깅 (KoBERT)
                         ↓
                    해설 사전 생성 (GPT-4o)
```

**금지**: 평가원 기출 원문 재게시 / 출판사 교과서 원문 무단 복제 / 수능 지문 인용

## 8. MVP 범위 (3개월) — 좁히기 원칙

| 항목 | 결정 |
|---|---|
| 학년·교과 | **중1 수학 1종**만 (출판사 1~2 종) |
| 문제 수 | **3,000건** (단원당 250 × 12 단원) |
| 유형 | 객관식 + 단답형 (서술형 제외) |
| AI 진단 | **룰 기반** ("동일 개념 3회 중 2 오답 → 약점") |
| 결제 | 무프리미엄 (일 5문제 무료 / 월 ₩9,900 무제한) |
| 학부모 모드 | 주간 리포트만 (대시보드는 Phase 2) |
| 게이미피케이션 | 스트릭·진척도 바만 (배지·랭킹 X) |

**제외 (Phase 2 이후)**: 푸시 알림, 라이브 과외 매칭, 영상 해설, 커뮤니티, 학원 모드
